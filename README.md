# 👕 Kaps Clothing Pack (KCP) - V2

Welcome to **V2** of the Kaps Clothing Pack (KCP)! This pack is the second installment in the series, continuing to bring a variety of new clothing options to your game or project.

---

## 📦 About This Release

This repository contains the second version (**V2**) of the Kaps Clothing Pack. It includes the continuation set of clothing assets.

## ⬇️ How to Get the Full Collection

The Kaps Clothing Pack is a continuing project spread across multiple versions. **V2 is a continuation pack.** To ensure you have access to **all** the clothing assets KCP has to offer, you will need to download **all four** currently available versions:

* **V1** (Initial Pack)
* **V2** (This Repository - Continuation Pack)
* **V3** (Continuation Pack)
* **V4** (Continuation Pack)

Please check the main KCP developer page or the relevant links to locate and download the V1, V3, and V4 packages.

---

## 🛠️ Installation & Usage

Installation is simple!

1. **Drag and Drop** the files into your game's directory.
2. **Ensure** you have also downloaded and installed **V1, V3, and V4** for the complete Kaps Clothing Pack collection.
